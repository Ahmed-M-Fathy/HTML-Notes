Question: Which two elements (that you've learned about so far) have optional closing tags?
Answer: The "Paragraph" element and the "List" element.

Question: When you leave off a closing tag, how does the browser know when to close that element?
Answer: There are two cases:

  1. If the closing tag is mandatory, the browser will not know when to close the element
  2. If the closing tag is Optional, the browser will know when to close the element
